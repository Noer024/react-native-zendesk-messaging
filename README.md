# react-native-zendesk-messaging

React Native Library for Zendesk Messaging

## Installation

```sh
npm install react-native-zendesk-messaging
```

## Usage

```js
import { multiply } from "react-native-zendesk-messaging";

// ...

const result = await multiply(3, 7);
```
